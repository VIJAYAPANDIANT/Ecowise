## 📖 Overview

**EcoWise** is a modern, AI-powered conversational assistant designed to empower users with the knowledge they need to live more sustainably. By providing clear, actionable advice on waste disposal, recycling, and responsible consumption, EcoWise helps bridge the gap between intent and action in the "Reduce, Reuse, Recycle" hierarchy.

Built with a focus on both functionality and aesthetics, EcoWise features a stunning **Glassmorphism UI** that creates a premium, immersive user experience.

## ✨ Key Features

- 🧠 **Smart Waste Recognition**: Instantly identify the correct disposal method for plastics, metals, organics, e-waste, and more.
- 🎨 **Modern Aesthetics**: A sleek interface featuring dynamic background animations and premium glass-like UI elements.
- 💬 **Interactive Chat**: A responsive conversational interface that provides real-time guidance.
- 🛡️ **Safety Awareness**: Built-in logic to identify hazardous materials and provide essential safety warnings.
- ♻️ **Sustainability First**: Prioritizes reduction and reuse over recycling to encourage true circular economy practices.

## 🛠️ Tech Stack

- **Frontend**: Pure HTML5 & CSS3
- **Logic**: Vanilla JavaScript (ES6+)
- **Typography**: Google Fonts
- **Design System**: Custom CSS Glassmorphism & Micro-animations

## 🚀 Getting Started

### Prerequisites
A modern web browser (Chrome, Firefox, Safari, or Edge).

### Installation
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/VIJAYAPANDIANT/Ecowise.git
    ```
2.  **Navigate to the project directory**:
    ```bash
    cd Ecowise
    ```
3.  **Launch the Assistant**:
    - Open `index.html` in your favorite browser.
    - *Alternatively, use a Live Server extension in VS Code for the best experience.*

## ⚠️ Disclaimer

EcoWise is an educational AI assistant demo. While it provides high-quality guidance based on general sustainability practices, **always verify instructions for hazardous or toxic materials** with your local sanitation department or relevant authorities.
